# React + TypeScript + Vite

## 初始化项目

```bash
npm init vite@latest //根据提示一步一步进行
```

## 安装依赖

```bash
npm install
```

## 启动项目

```bash
npm run dev
```

## 打包项目

```bash
npm run build
```

## 项目结构

```bash
├── public                  // 静态资源
├── src                      // 源代码
│   ├── assets              // 静态资源
│   ├── components          // 公共组件
│   ├── pages               // 页面
│   ├── router               // 路由
│   ├── store                // 状态管理
│   ├── styles               // 样式
│   ├── utils                // 工具类
│   ├── App.tsx               // 入口文件
│   ├── main.tsx             // 入口文件
│   └── vite-env.d.ts       // 环境变量
├── index.html              // 入口html
├── tsconfig.json           // ts配置文件
└── vite.config.ts          // vite配置文件
```
